1. System Architecture

Hastara delivers institutional-grade AI palm readings across iOS, Android, and web — unified through a Turborepo monorepo and a single PostgreSQL backend. Every reading synthesises three palmistry traditions using GPT-5 Vision and Claude Opus 4.7.

1.1 Technology Stack
LayerTechnologyVersionPurpose
MobileReact Native + ExpoSDK 51iOS + Android
WebNext.js App Router15.0.xSEO + checkout
APItRPC v1111.xType-safe RPC
DatabasePostgreSQL (Supabase)16Multi-region
Important — Gold callout box
All AI calls are server-side only. No API keys in client bundles. Subscription tier is enforced at database level via Row-Level Security policies.

Footnote / page header / footer text uses Muted Gray #666666 · Body text uses Charcoal #1A1A1A · All table and card borders use Gold Border #D4C9B0
